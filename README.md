## Installing

## Running

```
$ ./gradlew run
```

## Building

```
$ ./gradlew build
```

## Testing

To test all packages, run the following:
```
$ ./gradlew test
```

To test just server-side Java, run the following:
```
$ ./gradlew :server:test
```

## Built With

* [Gradle](https://gradle.org/) (or user the wrapper, recommended)
* [Node Package Manager(npm)](https://www.npmjs.com/get-npm)
