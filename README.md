## Installing

## Running

```
$ ./gradlew run
```

## Building

```
$ ./gradlew build
```

## Testing

To test all packages, run the following:
```
$ ./gradlew test
```

To test just server-side Java, run the following:
```
$ ./gradlew :server:test
```

To have more detailed output about tests, add the --info argument:
```
$ ./gradlew :server:test --info
```
